# DeskPilot（Windows 按鍵精靈工具）需求規格文件 v1.0

## 一、產品概述

DeskPilot 是一款 Windows 桌面自動化工具，提供使用者透過 GUI
操作介面，錄製與重播滑鼠與鍵盤行為，並支援視窗位置與尺寸鎖定，提升重複性操作效率。

## 二、產品目標

-   提供非工程師可使用的自動化工具
-   降低重複操作成本
-   支援展場、日常操作、自動流程執行
-   提供穩定的視窗控制能力

## 三、目標使用者

-   內部營運 / 行政人員
-   展場操作人員
-   需執行固定流程的使用者

## 四、系統平台

-   作業系統：Windows 10 / Windows 11
-   開發語言：Python
-   GUI 框架：PyQt6

## 五、功能需求

### 5.1 視窗控制模組（核心功能）

允許使用者選取指定視窗，並鎖定其位置與尺寸。

### 5.2 操作錄製模組

錄製滑鼠與鍵盤操作並轉為腳本。

### 5.3 腳本播放模組

支援單次、多次與循環播放。

### 5.4 GUI 介面

提供完整操作控制面板。

### 5.5 熱鍵控制

-   F6 播放
-   F7 停止
-   F8 錄製

### 5.6 資料儲存

使用 JSON 儲存腳本與設定。

## 六、系統架構

GUI → Controller → Modules

## 七、技術設計

-   win32gui（視窗控制）
-   pynput（錄製）
-   pyautogui（播放）

## 八、非功能需求

-   易用性高
-   低延遲
-   支援多解析度

## 九、限制

-   僅支援前景操作
-   遊戲可能受限

## 十、部署

使用 PyInstaller 打包為 exe

## 十一、開發階段

Phase1: 視窗控制 + UI Phase2: 錄製 Phase3: 播放 Phase4: 儲存
